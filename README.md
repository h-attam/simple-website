# Basit HTML Portföy Sitesi
Bu proje, sadece HTML kullanılarak hazırlanmış basit ve hızlı yüklenen bir portföy websitesidir.
Amaç, temel HTML yapısını kullanarak kişisel bilgilerimi, projelerimi ve iletişim bilgilerimi sade bir şekilde sunmaktır.

## Özellikler
Tamamen HTML kullanılarak geliştirilmiştir
Hafif ve hızlı yüklenen yapı
Temiz ve anlaşılır kod yapısı
Hakkımda bölümü
Projeler bölümü
İletişim bölümü

Not: Proje tek sayfa ise yalnızca index.html yeterlidir.

## Kullanılan Teknolojiler
- HTML5
- Herhangi bir framework, kütüphane veya stil dosyası kullanılmamıştır.
